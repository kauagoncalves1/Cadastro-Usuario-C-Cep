-- Cadastro de Usuário com CEP --

Formulário de cadastro de usuário com preenchimento automático de endereço via CEP, desenvolvido em HTML, CSS e JavaScript como parte da minha jornada de aprendizado.

-- O que o projeto faz --

O usuário preenche seus dados — nome, e-mail, senha e CEP — e o endereço é preenchido automaticamente consultando a API do ViaCEP. O formulário também valida todos os campos em tempo real antes de enviar.

-- Funcionalidades --

- Preenchimento automático de endereço pelo CEP (API ViaCEP)
- Validação de campos em tempo real
- Requisitos de senha com feedback visual (verde/vermelho)
- Mostrar/ocultar senha com ícone interativo
- Confirmação de senha
- Mensagens de erro amigáveis

-- O que foi usado --

- HTML — estrutura do formulário e semântica
- CSS — layout com Flexbox, estilização do card e estados visuais
- JavaScript — manipulação do DOM, eventos, validações
- Fetch API — primeira vez consumindo uma API real
- Font Awesome — ícones profissionais
- ViaCEP — API grtuita de consulta de endereço por CEP
